The Phishing Detection System is a machine learning-based application designed to identify phishing websites by analyzing URL and webpage features. The system extracts relevant attributes from website data and applies classification algorithms to determine whether a website is legitimate or malicious. This project demonstrates the use of data preprocessing, feature engineering, and machine learning techniques for improving cybersecurity and automated threat detection.

Features:

Detects phishing websites using machine learning models
URL and website feature extraction
Data preprocessing and feature engineering
Automated classification of websites as phishing or legitimate
Improves cybersecurity awareness and threat prevention

Technologies Used:

Python
Pandas
NumPy
Scikit-learn
Jupyter Notebook

Project Workflow:

Data collection of phishing and legitimate website URLs
Data preprocessing and cleaning
Feature extraction from URLs and website attributes
Training machine learning classification models
